> **Почему Oculo?** > От латинского *oculus* — глаз. Акцент на абсолютном «видении» доски. Это название подчеркивает главную фичу платформы: наш движок буквально насквозь видит позицию, замечает малейшие «зевки» и оценивает стратегию на несколько ходов вперед, чтобы обучать игрока.

**Oculo** — это не просто очередная браузерная игра «Четыре в ряд». Это образовательная SaaS-платформа для тренировки стратегического мышления, созданная на стыке хардкорной алгоритмики, современного дизайна и нейросетевых технологий. 

---

## ✨ Ключевые фичи (Почему это уровень «Великий»)

### 🧠 1. AI Coach & Нейросимволический анализ
Платформа не просто играет с вами, она вас **учит**. После (или во время) матча ИИ-тренер анализирует историю ходов и выдает контекстные советы на естественном языке, указывая на допущенные ошибки и упущенные «вилки». (Монетизируется через Pro-подписку).

### ⚡ 2. Bitboard-Оптимизация (Core Engine)
Большинство реализаций используют медленные 2D-массивы. Мы переписали движок, используя **64-битные целые числа (BigInt)**. Состояние доски кодируется в 49 битах. 
* Проверка победы выполняется за $O(1)$ через побитовые сдвиги (`>>` и `&`).
* Позволяет анализировать миллионы состояний в секунду.

### 🧵 3. Web Workers & Minimax
Тяжелый алгоритм **Minimax с Альфа-Бета отсечением** вынесен в изолированный фоновый поток браузера (`ai.worker.ts`). Результат: ИИ просчитывает ходы на глубину 5+ шагов, а UI (анимации, частицы, видео-фон) не тормозит ни на миллисекунду.

### 🎨 4. Эстетика Quiet Luxury & Саунд-дизайн
Строгий минимализм, неоновые акценты, пиксельные шрифты (*Press Start 2P*) и пастельные тона. В приложение интегрирован эмбиент-саундтрек (`music.ogg`) и кинематографичный видео-фон, создающие состояние глубокого фокуса (погружения).

### 🥚 5. Easter Eggs & Социальный слой
Подождите в главном меню 5 секунд... и вы найдете кое-что важное. *(You know the rules, and so do I! 🎵)*

## 🛠 Технологический Стек

* **Фронтенд:** Next.js (App Router), React 19, TypeScript
* **Стилизация:** Tailwind CSS, `clsx`, `tailwind-merge`
* **Состояние:** React `useReducer` + Custom Hooks
* **Алгоритмы:** Bitboards, Minimax Algorithm (Alpha-Beta Pruning)
* **Анимации:** `canvas-confetti`, CSS Transitions
* **Бизнес-модель:** Freemium (интеграция моков Stripe для Pro-статуса)
