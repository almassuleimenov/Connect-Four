# NeuroConnect4

**NeuroConnect4 — образовательная SaaS-платформа для тренировки стратегического мышления через соревновательные механики и AI-аналитику.**

## О проекте (Архитектура)

NeuroConnect4 представляет собой радикальный отход от традиционных браузерных симуляций настольных игр, предлагая полноценную SaaS-платформу.

Технологический стек:
*   **Фронтенд:** Next.js (App Router), React, Tailwind CSS
*   **Бэкенд:** Supabase (PostgreSQL, Realtime, Edge Functions)
*   **Платежи:** Stripe
*   **AI:** Интеграция LLM (OpenAI/Anthropic) для анализа

**Ключевые архитектурные решения:**
1.  **Серверная Авторитарность:** Логика игры полностью перенесена на сервер с использованием Supabase Realtime (Row Level Security), что делает невозможным читерство со стороны клиента и защищает киберспортивные матчи.
2.  **Гибридный AI Coach:** Мы объединили высокооптимизированный детерминированный алгоритм Minimax (с альфа-бета отсечением и битовыми досками, вычисляемый в Web Workers) с мощью современных LLM. Движок математически оценивает позицию и выявляет "зевки", а LLM переводит эти данные в эмпатичные человекочитаемые советы, исключая риск галлюцинаций в правилах игры.
3.  **Высокая производительность:** Использование битовых досок для представления состояния игры обеспечивает анализ миллионов узлов в секунду.

## Для кого это (Целевая аудитория и Ниша)

Продукт нацелен на любителей логических игр, желающих улучшить свои аналитические способности, а также на киберспортивное комьюнити, нуждающееся в надежной инфраструктуре для проведения локальных онлайн-турниров (с опорой на геолокационные лидерборды). Встроенная система скрытого проксирования IP-адресов позволяет динамически формировать локальные сообщества (например, "Топ-10 стратегов Алматы"), радикально повышая социальное вовлечение.

## Почему это ценно (Бизнес-потенциал)

Бизнес-модель строится на стратегии Freemium с подпиской Pro.
*   Бесплатный доступ к базовому функционалу (игра с ИИ начального уровня).
*   **Pro-подписка (Stripe):** Открывает доступ к глубокому анализу партий (AI Coach), отсутствию рекламы, детальной статистике и кастомным скинам. Настроен Stripe Customer Portal для самообслуживания (управление подпиской, смена карт), что снижает операционные расходы. Деградация услуг при отмене подписки автоматизирована через вебхуки и RLS-политики.

## Инструкции по запуску

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/neuro-connect-four.git
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Настройте переменные окружения:
   Создайте файл `.env.local` в корне проекта и добавьте следующие ключи:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   STRIPE_SECRET_KEY=your_stripe_secret_key
   NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_stripe_publishable_key
   STRIPE_WEBHOOK_SECRET=your_stripe_webhook_secret
   ```
4. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
5. Откройте [http://localhost:3000](http://localhost:3000) в браузере.
